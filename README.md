# 🤖 AI Chat — бесплатный чат с ИИ и генерация картинок

**Работает прямо в браузере, без API-ключей, без регистрации, без лимитов!**

[![Pollinations AI](https://img.shields.io/badge/Pollinations-API-7c5cfc)](https://pollinations.ai)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

---

## ✨ Возможности

- 💬 **Чат с ИИ** на базе Pollinations AI (модель openai)
- 🖼 **Генерация изображений** по текстовому описанию
- 📱 **Адаптивный дизайн** — отлично работает на телефонах и планшетах
- 🌙 **Тёмная / светлая тема**
- 💾 **Сохранение истории чатов** в localStorage
- 📝 **Переименование и удаление чатов**
- 🔍 **Поиск по чатам**
- 📋 **Копирование кода из сообщений**
- 💾 **Скачивание кода** как файл
- 🎨 **Подсветка синтаксиса** для более чем 100 языков
